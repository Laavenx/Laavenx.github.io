BattleChant

Unreal Engine Turn-based game in pre-release state. The game is made purely using Blueprints currently.



Features that I've implemented:

  • Grid-based system working with offset coordinates
  • Turn-based system integrated with GAS, supporting pathfinding, multilevel grids, turn management, and other turn-based mechanics (written in C++).
  • Top-down custom camera, which can also be moved around using events from grid system
  • Units that can be placed and anchored to the grid in the editor. They hold GAS abilities, have an Ability System Component, and display received damage via Gameplay Cues.
  • Grid UI displaying tile markers for movement range and enemy locationsm, displaying spline when hovering on the grid.
  • Adaptive cursor that changes based on the hovered tile.
  • Custom user interface, including ability selection per NPC, a quest slider, and an End Turn button
  • Dialogue system for quest events and in-game conversations
  • Designed with multiplayer support in mind.
