BattleChant

Unreal Engine Turn-based game in pre-release state. The game is made purely using Blueprints currently.



Some of the things that I've implemented:

  • Designed a system which makes it possible for a writer to be able to create their own dialogues. I accomplished this by creating a FRowType which can hold Text, SoundWave and character portrait as Texture2D. This then can be loaded sequentially using DataTable by the Dialogue UI Blueprint.
  • Changed Turn Based System to give priority of action consecutively for example: player takes action with all of their units and then enemy does the same.
  • Player being able to freely pick which unit to take action with during consecutive turns.
  • Implemented combat stats and calculations according to GDD
