BFME Reforged

BFME Reforged is a Fan project that I used to contribute to. It is a Unreal Engine RTS game incorporating base building, unit production and combat.

I have learned Unreal Engine to be able to contribute to this project and then I progressed to solving more complex problems.

This game is written to support thousands of units at 60fps. I have spent a lot of time figuring out how to utilize unreal systems to be able to do so. The units are contained within a Battalion which has a set formation. They work similary to Boids, their actions are dynamically calculated in the mass system using a set of modular "rules" for each entity.

Some things I've implemented:

  • Playing select sounds, unit movement sounds, spawn sounds.
  • Performant way of selecting units
  • Octree for buildings to cull what Mass Entity System is iterating over.
  • Boid avoidance and collision against buildings using my own generated 2D collision.
  • Improved existing Boid movement by refactoring it to closely follow given path making it execute "IsBoidLate" logic to accelerate towards its Battalion. Added "LostBoid" logic which makes the unit go at full speed through it's own, newly created control points in order to reach the Battalion ( It's an edge case that shouldn't happen very often ). I also added a StateTree Task for when Battalion spawns and exits building (has to have collision turned off and reach a proper spot).
  • Builder ability to construct new buildings






One of the bigger things I've managed to solve is how to generate a 2d collision ( Units work on a 2D plane ) for entities, to make it cheaper. I've done so by taking all of the vertices of a 3D entity, clamping a range and removing all unneeded vertices and then using 2D convex hull alghoritm to trim all the innermost points. Produced 2D polygon can be shrunk or expanded by using imported part of Clipper2 library. All of it is configurable within editor.



I've collaborated with an audio designer by creating an easy way for him to add different sounds to an entity.
