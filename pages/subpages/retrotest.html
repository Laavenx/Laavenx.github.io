Retro Test

A project developed to demonstrate creation of 3D platformer using GAS and custom movement in UE5.



Full implementation of GAS combat, using AI, reactive UI with healthbar and a Cooldown for Ability, Knockback effect from enemy. The slime character also has custom Blob Shadow.



When montage event fires, NPC does a sweep and applies damage and knockback.



Custom movement which changes gravity using timelines and applies impulse, replicated in multiplayer.
