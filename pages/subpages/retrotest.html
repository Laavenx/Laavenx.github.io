Retro Test

A project developed to demonstrate creation of 3D platformer using GAS and custom movement in UE5. https://github.com/Laavenx/RetroTest
  • Custom MovementComponent changes gravity based on timeline curve per tick, for jumping and falling.
  • Player input is checked for preemtive jump input, to perform auto-jump so that player can chain jumps more easily
  • Array of Structs storing angle and magnitude, determines how consecutive jumps behave.
  • Adding impulse is done using MultiCast RPC to keep it in sync with the server


  • AttributeSet keeps track of received damage and responds accordinly applying knockback or destroying the owner
