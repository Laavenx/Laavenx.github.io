Samsara Saga

Upcoming mmorpg unreal engine 5 game, that I'm working on currently.

I'am working on mmorpg gameplay systems using unreal Replication and GAS. The game uses Paper2D sprites which are manually rotated towards camera, they have 8 axis direction.



OWS plugin allows us to automatically create instances of dedicated server, and to setup and communicate with Database.

Features that I've implemented:



  • Character using Paper2D, with custom logic to render entities facing towards Camera.
  • Inventory system.
  • Player controls: target selection, camera.
  • NPC Character behaviours using State Tree.
  • Server Authoritative Gameplay Ability System.
  • Networked Code for both client and server.
  • Client settings sent to server.
  • Implement OWS features like login and portals.
