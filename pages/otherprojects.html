SEDB

A website made using Svelte framework that I've made in the past, serving as a source of information about entities from Space Engineers. The website uses SQlite to store information about entities. The database was created using a python script, converting the game .xml files.



Server script loads data from SQlite using sequelize ORM. The data gets passed to appropriate component, where the it gets injected;



The project implements lazy loading of only visible items, which makes it friendly for low bandwidth connections.
